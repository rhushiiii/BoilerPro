# BoilerPro SaaS Starter

Production-ready Next.js 14 starter with App Router, Tailwind CSS, and shadcn-style UI primitives.

## Features

- App Router with grouped routes for marketing, auth, and dashboard
- Tailwind CSS configured with design tokens
- Reusable UI components in `components/ui`
- Responsive navbar and dashboard sidebar
- TypeScript-ready folder structure (`app`, `components`, `lib`, `hooks`, `types`)
- Environment template in `.env.example`

## Routes

- `/` Landing page
- `/pricing` Pricing page
- `/dashboard` Dashboard
- `/login` Login
- `/signup` Signup

## Run locally

```bash
npm install
npm run dev
```
